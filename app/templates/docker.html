How to setup webpage using Docker Compose

  1. If you haven't already, make a Dockerfile (steps in apache page)
  2. Enter touch docker-compose.yml to create a file that will deploy, combine, and configure multiple docker-container at the same time
  3. Enter vi docker-compose.yml
  4. Make sure the docker-compose.yml file has the following text:
    5. version: "3.2"
    services:
       apache:
        build: .
        ports:
         - "8080:80"
        volumes:
         - ./public_html/:/usr/local/apache2/htdocs/
  5. Save and exit vi.
  6. In your terminal enter docker compose up -d to run the docker compose file, which in this case will start an apache server.
  7. Now open up your favorite browser and goto this url localhost:8080 to see your site.
  8. To stop the docker container, enter docker compose down.
